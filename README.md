## 介绍

是一个程序员技术学习社群，目前组织以前端为主的技术交流。

加入标准：**至少半年内做一场技术分享或者组织一次peer活动**。

发起分享活动，创建一个issue链接，便于知识沉淀。

修改文档申请加入项目或者直接PR。
